# Noobify your music experience
Noobify ♪ - Music Player 
```Uses ExoPlayer as Media Player```

- listen to the song✅ || play/pause the song✅ || automatically navigate to next song on completion of current song✅ || Playing songs in the background state of the app✅

- audio player notification like spotify(In lockscreen as well)✅ || Speed controls✅
- Add pagination using the same API infinitely✅

- Thoughtful UI and Smooth Animations ✅ || Loading Animation✅

- Kotlin✅ || Live Data ✅ || Architecture: MVVM✅ || Dependency Injection: Dagger- Hilt

- Clean, modular, reusable, documented and easily readable code✅✅
- Appropriate naming and Validations/Error Handling✅
- Proper commits✅
- Followed best practices✅
- Multithreading/Concurrency✅

# Set Up Code
 ``` 
 git clone https://github.com/msharibahmed/noobify.git
```
